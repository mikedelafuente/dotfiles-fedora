# Neovim Cheat Sheet - Quick Reference

**Leader Key:** `<Space>` (press space to see all available commands)

---

## üöÄ Getting Started

### Opening Neovim
```bash
nvim                    # Open Neovim
nvim filename.txt       # Open specific file
nvim .                  # Open in current directory
```

### Exiting Neovim
- `:q` - Quit (fails if unsaved changes)
- `:q!` - Quit without saving
- `:w` - Save file
- `:wq` or `ZZ` - Save and quit
- `<Ctrl-s>` - Quick save

## üìù Basic Editing

### Modes
- `i` - Insert mode (before cursor)
- `a` - Insert mode (after cursor)
- `A` - Insert at end of line
- `o` - Insert new line below
- `O` - Insert new line above
- `<Esc>` - Return to Normal mode
- `v` - Visual mode (select text)
- `V` - Visual Line mode

### Movement
- `h` `j` `k` `l` - Left, Down, Up, Right
- `w` - Next word
- `b` - Previous word
- `0` - Start of line
- `$` - End of line
- `gg` - Top of file
- `G` - Bottom of file
- `{` `}` - Previous/Next paragraph
- `<Ctrl-u>` - Page up
- `<Ctrl-d>` - Page down

### Editing
- `x` - Delete character
- `dd` - Delete line
- `yy` - Copy (yank) line
- `p` - Paste after cursor
- `P` - Paste before cursor
- `u` - Undo
- `<Ctrl-r>` - Redo
- `.` - Repeat last command

## üîç Finding Things (Telescope)

### File & Text Search
- `<Space>ff` - Find files (fuzzy search)
- `<Space>fg` - Live grep (search in files)
- `<Space>fb` - Find buffers (open files)
- `<Space>fh` - Help tags
- `<Space>fr` - Recent files
- `<Space>fc` - Find word under cursor
- `<Space>fk` - Find keymaps

### Inside Telescope
- `<Ctrl-n>` / `<Ctrl-p>` - Next/Previous result
- `<Enter>` - Open file
- `<Esc>` - Close Telescope
- `<Ctrl-x>` - Open in horizontal split
- `<Ctrl-v>` - Open in vertical split

## ü™ü Windows & Buffers

### Window Navigation
- `<Ctrl-h>` - Move to left window
- `<Ctrl-j>` - Move to bottom window
- `<Ctrl-k>` - Move to top window
- `<Ctrl-l>` - Move to right window

### Window Management
- `<Ctrl-Up>` - Increase window height
- `<Ctrl-Down>` - Decrease window height
- `<Ctrl-Left>` - Decrease window width
- `<Ctrl-Right>` - Increase window width
- `:split` or `<Ctrl-w>s` - Horizontal split
- `:vsplit` or `<Ctrl-w>v` - Vertical split

### Buffer Navigation
- `<Shift-h>` - Previous buffer
- `<Shift-l>` - Next buffer
- `<Space>bd` - Delete buffer

### Diagnostics (Errors/Warnings)
- `[d` - Previous diagnostic

- `<Space>d` - Show diagnostic details
- `<Space>q` - Open diagnostic list

## üîß Useful Commands

### File Operations
- `:e filename` - Edit/open file
- `:w` - Save file
- `:w filename` - Save as
- `:r filename` - Read file into current buffer

### Search & Replace
- `/pattern` - Search forward
- `?pattern` - Search backward
- `n` - Next search result
- `N` - Previous search result
- `:%s/old/new/g` - Replace all in file
- `:s/old/new/g` - Replace in current line

### System
- `:checkhealth` - Check Neovim health
- `:Mason` - Open LSP server manager
- `:Lazy` - Open plugin manager
- `:LspInfo` - Show LSP status

## üíª Terminal

### Opening Terminal
- `:terminal` or `:term` - Open terminal in current window
- `:split | terminal` - Open terminal in horizontal split
- `:vsplit | terminal` - Open terminal in vertical split

### In Terminal Mode
- `<Ctrl-\><Ctrl-n>` - Exit terminal mode (back to normal mode)

## üî• Pro Tips

1. **Press `<Space>` and wait** - which-key will show you all available commands
2. **Use `K` over symbols** - Shows documentation instantly
3. **Use `gd` to jump to definitions** - Navigate code quickly
4. **Use `<Space>ff`** - Fuzzy find files faster than file explorer
5. **Use `<Space>fg`** - Search text across all files
6. **Press `?` in any plugin** - Most plugins have contextual help
